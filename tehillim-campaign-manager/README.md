# Tehillim Campaign Manager 2.6.3

גרסה מסודרת בכמה קבצים, עם תאימות מלאה לשורטקודים ולנתונים של הגרסאות הקודמות.

## מה חדש
- תיקון בחירת כמה פרקים: מעבר אוטומטי לפרק הבא אחרי סימון השלמה.
- סדר טופס משופר: בחירת פרק לפני שם/אימייל/טלפון.
- שם, אימייל וטלפון הם רשות ומשמשים לקבלת הפרק ותזכורות.
- הרחבת `[tehillim_data]` עם פרמטרים נוספים, prefix/suffix ותנאי if_less.
- מדריך שימוש מורחב בלוח הבקרה.

## לאחר התקנה
להיכנס ל: הגדרות → קישורים קבועים → שמירת שינויים.
