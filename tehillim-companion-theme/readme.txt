=== Tehillim Companion ===

A lightweight, RTL-first, accessible block theme that pairs with the
Tehillim Campaign Manager plugin.

The PLUGIN holds all logic and data (campaigns, chapters, ambassadors,
subscriptions, ads). This THEME only provides a clean, ready-to-run look,
so switching themes never touches your data.

== Installation ==

1. Install and activate the "Tehillim Campaign Manager" plugin.
2. Upload this theme to wp-content/themes/ and activate it.
3. Settings → Permalinks → Save (to register the campaign URLs).

Campaign and prayer pages render automatically via the plugin's
templates; no extra setup is needed.
